# 恋爱人格测试网站：普通分享 + 可选配对版

## 新增功能

结果页有两个主要按钮：

### 1. 分享给朋友一起测
这个是普通分享。
朋友打开后只做自己的测试，只看到自己的结果。
不会和你配对。

### 2. 邀请朋友配对分析
这个是配对分享。
朋友打开后完成测试，会自动生成你们两个人的匹配报告。

## 双人匹配报告包含

- 总体匹配度
- 双人维度雷达图
- 各维度匹配打分
- 双方优势比对
- 双方短板比对
- 潜在摩擦点
- 相处建议
- 可生成图片分享

## 上传 GitHub

把以下文件全部覆盖你仓库中的同名文件：

- index.html
- style.css
- questions.js
- app.js
- README.md

Vercel 会自动重新部署。
